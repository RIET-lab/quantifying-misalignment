# Quantifying Misalignment Between Agents: Towards a Sociotechnical Understanding of Alignment

This repository contains the code and experiments for the paper "Quantifying Misalignment Between Agents: Towards a Sociotechnical Understanding of Alignment" by Aidan Kierans, Avijit Ghosh, Hananel Hazan, and Shiri Dori-Hacohen.

## Requirements

You can install the required packages using:

```
pip install -r requirements.txt
```

## Repository Structure

- `src/`
  - `multiagent_misalignment_model.py`: Contains the core classes and functions for the misalignment model.
  - `experiments.ipynb`: Includes the simulation experiments and plotting functions used in the paper.
